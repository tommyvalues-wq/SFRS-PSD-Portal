# SFRS Portal Node 24

No native SQLite package. Works on Node 24 without Python.

## Run
```cmd
npm.cmd install
npm.cmd start
```
Open http://localhost:3000

## Local demo login
Use **Demo Sign In** to access without Roblox while testing.

## Roblox login
Copy `.env.example` to `.env`, add Roblox OAuth details, then use Continue with Roblox.
